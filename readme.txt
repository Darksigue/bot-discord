Requirements:
* yarn
* node

OBS: In your first time using, you've to install the dependencies of the project with:

yarn

Terminal commands:
* node deploy-commands.js
* yarn start

Bot Commands:

/task

- Creates a task in a specific channel based on the channel ID

/pagar

- Returns a status of payment message

/avaliar

- Gives a review of a task created mentioning the user that executed
